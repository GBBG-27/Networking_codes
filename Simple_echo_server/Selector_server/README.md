## Description

This is a simple echo server that echoes back every data

sent to it.When the script starts,you will be prompt to enter

an ip address or a hostname with a port number,next the server 

will start to listen on that specific address.When data is received

(sent from the other end),is will be printed out and when the data

is echoed back,it will be also be printed out.A connection can be 

established for a single client at a time and when the connection is

closed from one end,the other end is also disconnected.

## Usage

##### - Make sure you have python installed (on most unix based computers python is pre-installed)
##### - Download the script or copy it into a python file on ypur computer
##### - Run the script (on the terminal,run 'python select_server.py' or with the file on your computer)
##### - Enter the hostname then click enter
##### - Next,type in the port number and click enter
##### - It start's listening,when a connection is made,that specific address will be printed out
##### - When data is received,it will be printed out and also printed out when echoed back as well
