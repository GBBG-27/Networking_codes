## Description

This is a simple script acting as a client 

connecting to a server.When the file is executed,

you will be prompt to enter an IP address,then a 

port number [ make sure a server is listening to that address ].

The next thing you will do is to sent data to the server,

type in the data and click enter.When an error occurs,it will be 

printed out,even if the server disconnected.

## Usage

#### - Make sure you have python installed on your computer
#### - First make sure a server is listening on an address
#### - Run the script in the terminal [ run python CLient.py  ] and click enter
#### - Next,type the hostname of the server and click enter
#### - Next,type rhe port number of the sever and click enter
#### - When the client connects,you will be prompt with two greater than sign [wait till you see two greater than sign] 
#### - You can then type the data to be sent and click enter
#### - When the server responds,the responds will be printed to the terminal [be patient,data might come or an error may occur]
#### - After that responds is printed out,you will be asked again to enter another data
